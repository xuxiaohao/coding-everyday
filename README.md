编码每日练习

## 提交规范

‌feat‌: 新功能或页面。
update: 更新功能或页面。
‌fix‌: 修复bug或解决冲突。
‌docs‌: 文档更新。
‌style‌: 代码格式调整（不影响代码逻辑）。
‌refactor‌: 代码重构（不添加新功能也不修复bug）。
‌perf‌: 性能优化。
‌test‌: 增加测试用例。
‌chore‌: 其他杂项事务，如更新构建工具等。
‌build‌: 构建系统或外部依赖项的变更。
‌ci‌: 持续集成配置的变更。
‌revert‌: 回滚到上一个版本。
